# Solução_Menu_Calculos
## Projeto Calculadora
  O projeto calculadora foi feito pra um exercicio escolar
e conta com estas funções:
- Somar/adição
- Subtrair/subtração
- Dividir
- Multiplicar
- Comparar números
- Par e ímpar
